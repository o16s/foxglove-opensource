## Installation

octaview Studio is available as a desktop application or can be self-hosted.

## Self-hosting

octaview Studio can be self-hosted using Docker:

```sh
docker run --rm -p "8080:8080" ghcr.io/octaview/studio:latest
```

octaview Studio will be accessible in your browser at [localhost:8080](http://localhost:8080/).
